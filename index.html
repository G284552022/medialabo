「Webコンテンツ作成」の成果

本テーマの学習目的

htmlやcss,JavaScriptを使い実際に自分でwebページを作れるようになることが目的です。それぞれがhtmlとリンクしてイベント処理や URLをさらに加えて使いやすく、見やすいものになるように考えてページを作ることも目標です。

プロジェクト課題について

HTML, CSS, JavaScript のページが反映されているページを作る。2回検索をしたときに1回目の検索結果を削除しているようにプログラムする。 AJAX通信を使って検索URLを貼って検索キーを設定して全てのデータが適した表示になるようにプログラムする。 検索機能にプルダウンメニューを採用して分かりやすく使いやすいようにする。検索にヒットした項目がすべて表示されるようにfor文などをつかしました。 cssで最後に装飾をして学んできたものをいかして作りました。

プロジェクト課題で作成したページ: project.html

数当てゲーム

課題4-1:数当てゲームのページを改良する。数当てゲームの結果をコンソールではなく，ページに表示するように 修正します。 予想のメッセージをconsole.log() ではなく HTML に出力するように書き換えます。 HTMLに「4」を予想する処理を4回繰り返す出力をします。

課題5-1: 数当てゲームを完成させる。htmlの最初のp要素の中のテキストを削除してラベル、テキストボックス、回答ボタンを設置する 回答ボタンのクリックイベントハンドラの登録とイベントハンドラの定義を行います。yosoの部分をテキストボックスの数になるように変更する。 デバックを行い完成しているか確認をします。

最終的に、3回入力するまでに正解できれば正解の判定が出て、3回以上間違えると答えが表示されて、4回以上回答するとゲームが終わっていることを 表示するものになるようなプログラミングを作る課題。

感想

まずHTMLについて学んでいきました。初めは、全くwebページの作り方が想像できなかったものの基礎を学んだことによってイメージがしっかり持てたので良かったです。 HTML 文書の基礎やリストやハイパーリンクはかなり簡単にやることができました。しかしさらに様々な要素を覚える必要があることに驚きました。 その中で一番画像の貼り付けには苦戦しました。お手本と少し違っていると思ったら画像とは違う段落の場所が間違えていて、細かさと難しさを学びました。 作業をしている間に自分の使いたいものを選んで試行錯誤できるという楽しさも感じられました。CSSを学びはじめてさらにできることが広がったのを感じました 。色や段落、箇条書きなどを少し付けることによってもページがガラッと変わるところは面白かったです。 文字に影をかけるような作業は理解に時間がかかってしまい難しかったです。JavaScriptの基礎を学びはじめた時はHTMLと混ざってしまって慣れるのに時間がかかりました。 コンソールを上手く使って作業するのにも慣れが必要でとても難しかったです。数当てゲームは初めに言われた時、本当に作れることが想像できなくて不安になりました。 デバックの部分でjavaのありがたみを感じられました。自分でどんなエラーなのか考えて紐解くことにかなり時間がかかって、思いついたものを片端から試して頑張りました。 DOMとJavaScriptのところから今まで習ったものが混ざりはじめて追いつくのがやっとになってしまい苦戦しました。特に顔の画像を３つとも表示するのが形の勘違いもあり、 １つしか表示できず大変でした。イベント処理は今回学んできた中で一番楽しんで作業ができました。 Webページに動きが一気に出てきて普段よく見るものも多くありできることが広がったのを感じました。 でも、イベント処理もたくさんの種類があって同じ形で書くところや違う形で書くところを抑えながら使い分けるのは難しかったです。 数当ての仕上げはとても苦戦しました。コンソール表示をページ自体に表示されるように変えるのは想像以上に頭の中で混乱してしまいました。 ボタンを押すfunctionの部分の}とじを最後にするのに中々気づけず何回もやり直しました。最後完成した時は自分に作れた驚きと嬉しさがありました。 最後にプロジェクト課題で習ったことを使って完成させました。URLからデータを上手く取り出して表示するのはとても苦戦しました。 全体を通して初めてで難しかったがだんだんと慣れてこられたので自信に繋がりました。これからも頑張って取り組みたいと思いました。
