毎回の実験で作成したページ

第1回 WWW と HTML

今回は HTML, CSS, JavaScript の基本と、 Web ページを構成する方法を学びました。 HTML 文書の基本を試しながらリストやハイパーリンクなどのさまざまな要素を使って課題のホームページをつくりました。レポートになるページも作成しました。

演習の成果

  • 演習1-3:HTML文書の作成
  • 演習1-4:HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

CSSによるホームページの見た目の設定することを学ぶ。例えば箇条書き、使用する要素についてやリンクの色設定などの基礎をはじめ、 スタイルシートの違いなどの学習。さらにウェブブラウザ説明ページの装飾演習を通して画像やレイアウトなどたくさんのCSSのプロパティに触れて使い方を学びました。

演習の成果

  • 練習問題2-3

第3回 JavaScript の基本

Web ページ上で複雑な機能を実現するためにJavaScriptを基礎から学んでいく。 FirefoxでHTMLファイルを開きコンソールを使った操作、 JavaScriptとjavaの違いを演習をしながら学んでいく。変数宣言や構文、デバックなどを合わせて学習しました。

演習の成果

  • 数当てゲーム
  • 練習問題3-6

第4回 DOM と JavaScript

DOM と JavaScript を組み合わせることによってWeb コンテンツの内容を読み取ったり， コンテンツに変更を加えたりする処理ができます。要素の検索、テキストの変更、span 要素とは、CSSプロパティの変更 を実際に演習しながら使い方を学びました。

演習の成果

  • 練習問題4-2から4-5

第5回 イベント処理

イベント処理の実現方法を演習しながら学びました。ボタンをクリックしたら文字色を変更するものイベントハンドラの登録、イベントハンドラの定義 を行いしました。イベントの様々な種類についてもふれました。それらを踏まえて数当てゲームとプロジェクト課題を行いました。
